# 🍝 Spaghetti Website

A simple, beautiful website all about spaghetti - Italy's most beloved pasta!

## About

This website celebrates the wonderful world of spaghetti, featuring:
- The history of spaghetti
- Different types of spaghetti
- How to cook perfect spaghetti
- Fun facts about spaghetti

## Built With

- React 18
- TypeScript
- CSS3

## To Run Locally

```bash
npm install
npm start
```

## Deployment

This project is set up for easy deployment on Vercel.
